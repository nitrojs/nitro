# Zeabur

> Deploy Nitro apps to [Zeabur](https://zeabur.com).

**Preset:** `zeabur`

:read-more{title="Zeabur" to="https://zeabur.com"}

::note
Integration with this provider is possible with [zero configuration](/deploy/#zero-config-providers).
::

## Deploy using git

1. Push your code to your git repository (currently only GitHub is supported).
2. [Import your project](https://zeabur.com/docs/get-started) into Zeabur.
3. Zeabur detects that you are using Nitro and enables the correct settings for your deployment.
4. Your application is deployed!
